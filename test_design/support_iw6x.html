IW6X Client Support

Below are guides to help you with any client issues you may have with IW6x.

IW6x Client Install Guide

If you do not own the game on Steam

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The IW6x Full Game torrent
  • The IW6x Client

Getting Started:

  1. Download the torrent which contains the full game including the DLC content.
  2. Move the folder to a location like C:\Games\IW6x.
  3. Place the IW6x.exe file you downloaded from GitHub and place in the IW6x folder.
  4. To start the game, double-click iw6x.exe and a menu will appear asking you to choose between The Singleplayer or Multiplayer. Choose Multiplayer and the game should lauch.

To change your name, open the in-game console (press the ~ key) and type /name yourname.

To unlock all, open the in-game console (press the ~ key) and type /unlockall.

Custom classes are made in the Private Match section.

IW6x DLC Install Guide

If you do own the game on Steam, and want DLC.

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The IW6x DLC torrent
  • The IW6x Client

Getting Started:

  1. Download the torrent which contains the DLC content.
  2. Move the completed download into your Steam Ghosts Folder.
  3. Place the iw6x.exe in the game folder.
  4. To start the game, double-click iw6x.exe and a menu will appear asking you to choose between The Singleplayer or Multiplayer. Choose Multiplayer and the game should lauch.

IW6X Server Support

Below are guides to help you with any server issues you may have with IW6x.

IW6x Server Install Guide

Requirements:

  • The Visual C++ 2010 Redistributable Package and the Visual C++ 2015-2019 Redistributable Package (These are included in the IW6x Dedicated Server torrent linked below)
  • Visual Code, Sublime Text (You may use whatever editor you choose but we don't recommend Windows Notepad)
  • Torrent client - (qBittorrent is recommended)
  • Computer or VPS that's online 24/7 with a decent connection that has average or above specs (We strongly advise a VPS)
  • Some technical knowledge / background knowledge of computers

Server Setup

  1. Download the IW6x Dedicated Server Full torrent (See Requirements Section) and place it in a location of your choosing on your VPS or PC
  2. Download the IW6x client from the X Labs GitHub and extract the .exe into your server folder. (Get the .zip package)
  3. Download the DedicatedServer.bat and/or the ExtinctionDedicatedServer.bat from GitHub or from the files attached below. Place the .bat files in the serer folder.
  4. Download the server.cfg file from GitHub and place it in the IW6x sub-folder.
  5. Carefully review and edit the configuration settings file to your liking. Give it a hostname, gametype, and remove/add maps that you want on the rotate.
  6. Save the server.cfg file you just edit after you are done.
  7. Forward the TCP and UDP port (27016). If you changed the default port or add a server you may need to open addition ports (27017, 21018, etc.)
  8. Run the DedicatedServer.bat or ExtinctionDedicatedServer.bat and the server should begin to load.

Extinction Mode Server

If you want to set up an Extinction server, you must do the following:

  1. Add +aliensMode 1 to your .bat file or launch parameters.
  2. Adjust your map rotation to use the Extinction maps. Example: set sv_maprotation "gametype aliens map mp_alien_town map mp_alien_armory map mp_alien_beacon map mp_alien_dlc3 map mp_alien_last"

Known Issues

  1. Currently, g_gametype in server.cfg does not work. Game type must be set in the map rotation. Example: set sv_maprotation "gametype war map mp_prisonbreak map mp_dart
  2. The port cannot be set manually. The server will run on the default port of 27016. Should you decide to set up a second server, the server should automatically assign itself the next highest port.
  3. Currently, there is a bug that crashes dedicated servers. We suspect it is related to a map change. If you are a server owner, you can try extending your matches to infinity by doing scr_war_timelimit 0; scr_war_scorelimit 0. This will prevent TDM matches from ending and should thus be a workaround for the bug (if you are hosting other gamemodes, replace war accordingly). The X Labs team is working on a fix as fast as possible.
  4. Other issues may exist that are not covered here. This is an initial release, so be aware that you might experience difficulties running dedicated servers for IW6x.

Server Administration Software

There is no third-party admin tools at the moment. IW4MAdmin will add support at a later date.
