IW6x Client Install Guide

If you do not own the game on Steam

Requirements:

  • A Torrent Client. Recommended client qBittorrent
  • The IW6X Full Game torrent
  • The IW6x Client

Download the torrent which contains the full game including the DLC content. Move the folder to a location like C:\Games\IW6x. Place the IW6x.exe file you downloaded from GitHub and place in the IW6x folder. To start the game, double-click iw6x.exe and a menu will appear asking you to choose between The Campaign or Multiplayer. Choose Multiplayer and the game should lauch.

To change your name, open the in-game console (press the ~ key) and type /name yourname.

Custom classes are made in the Private Match section.
